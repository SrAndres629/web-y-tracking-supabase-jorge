# Gestión de Tipografía

## Goal
Mantener la elegancia y legibilidad del sitio mediante el uso estricto de las fuentes Playfair Display e Inter.

## Instructions
1. **Jerarquías**:
   - Encabezados: `Playfair Display` (Serif).
   - Cuerpo de texto: `Inter` (Sans).
2. **Modificación**:
   - Los ajustes se realizan en [`typography.css`](file:///home/jorand/antigravityobuntu/static/design-system/tokens/typography.css).
   - Asegura que los tamaños sigan la escala definida (Major Third Scale).

## Constraints
- No importar fuentes externas sin autorización; usar Google Fonts ya configuradas.
- Mantener el espaciado entre letras (`tracking`) definido para cada nivel.

# Mobile Strategy Reference - Jorge Aguirre Flores

## 1. The Thumb Zone Map
Diseñamos para que el usuario pueda completar la acción principal con una sola mano.

- **Green Zone (Natural)**: Parte inferior y central. Aquí deben ir los CTAs principales (WhatsApp, Agendar).
- **Yellow Zone (Stretched)**: Parte superior-central. Para navegación secundaria.
- **Red Zone (Hard to Reach)**: Esquinas superiores. Evitar colocar elementos críticos aquí.

## 2. Fitts' Law for Mobile
"El tiempo para adquirir un objetivo es una función de la distancia y el tamaño del objetivo."

- **Acción**: Aumentar el tamaño de los botones de conversión a un mínimo de **48px de alto**.
- **Acción**: Separar elementos clicables con al menos **8px de gap** para evitar clics accidentales.

## 3. Cognitive Load Reduction
En móvil, el usuario escanea rápidamente.

- **Hierarchy**: Título (Heading) -> Beneficio (Bullet) -> Acción (Button).
- **Negative Space**: No tengas miedo del espacio en blanco; ayuda a enfocar la atención en el CTA.

## 4. Mobile Technical Standards
- **Font-sizes**: `text-base` (16px) es el estándar.
- **Line-height**: Mantener un `leading-relaxed` (1.5) para legibilidad en pantallas pequeñas.
- **Tap Highlights**: Eliminar el tap highlight gris predeterminado en móviles para una sensación más premium (`-webkit-tap-highlight-color: transparent`).
